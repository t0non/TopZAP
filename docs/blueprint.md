# **App Name**: WhatsConnect

## Core Features:

- Painel de Controle: Exibe métricas chave e uma visão geral do desempenho da campanha WhatsApp.
- Gerenciamento de Contatos: Permite que os usuários importem, organizem e segmentem contatos para mensagens direcionadas.
- Criação de Campanhas: Permite que os usuários criem e agendem campanhas WhatsApp com mensagens personalizadas.
- Histórico de Campanhas: Fornece um histórico detalhado de campanhas passadas, incluindo status de entrega e métricas de engajamento.
- Otimização de Mensagens com IA: Utiliza IA generativa para sugerir melhorias no conteúdo da mensagem, maximizando o impacto e a entregabilidade; usa raciocínio para decidir se alguma informação sugerida melhora o texto.

## Style Guidelines:

- Cor primária: Verde do WhatsApp (#25D366) para ações primárias e branding.
- Cor de fundo: Cinza claro (#F0F2F5) para uma sensação limpa e moderna.
- Cor de destaque: Um tom ligeiramente mais escuro de verde (#128C7E) para estados de foco e destaques sutis.
- Fonte do corpo e título: 'Inter', um sans-serif estilo grotesco, para uma experiência moderna e legível.
- Use ícones simples e claros de uma biblioteca como FontAwesome ou Material Icons, alinhados com a estrutura de navegação (por exemplo, ícone de casa para Painel de Controle).
- Navegação na barra lateral com rótulos e ícones claros, usando o grid Tailwind CSS para layouts responsivos.
- Animações sutis para estados de carregamento e transições para melhorar a experiência do usuário.
